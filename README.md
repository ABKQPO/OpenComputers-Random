# OpenComputers Game

A terminal-based Snake game for the OpenComputers mod in Minecraft.

---

## Requirements

* OpenComputers computer
* Internet Card (required for downloading)
* Screen + GPU

---

## Download & Install

Use the built-in `wget` command to download the program:

